데이터 바인딩

개요 : Vue.js 에서 컴포넌트의 데이터와 템플릿을 연결하는 핵심 기능으로 이를 통해 동적으로 데이터를 화면에 표시하고 사용자의 입력과 상호작용을 관리할 수 있음

1. 문자열 데이터 바인딩

- 개요 : 텍스트 데이터를 HTML 요소에 바인딩하여 화면에 표시함
- 구현 방법 : {{ }} (Mustache 태그)를 사용하여 데이터를 텍스트 형태로 렌더링함

2. 원시 HTML 데이터 바인딩

- 개요 : html 코드를 포함한 문자열을 화면에 렌더링할 때 사용함
- 구현 방법 : v-html 지시자를 사용하여 html 을 원시 html로 렌더링 함

3. form 입력 데이터 바인딩

- 개요 : 폼 입력과 같은 인터랙티브 요소의 데이터를 동적으로 관리함.
- 구현 방법 : v-model 지시자를 사용하여 양방향 데이터 바인딩 설정함

4. 속성 바인딩

- 개요 : html 속성을 동적으로 바인딩 함
- 구현 방법 : v-bind 지시자 또는 : 축약형을 사용하여 속성에 데이터 바인딩 함

5. 클래스 바인딩

- 개요 : 요소의 클래스를 조건에 따라 동적으로 추가하거나 제거함
- 구현방법 : 객체 또는 배열 구문을 사용하여 클래스를 바인딩 함

6. 인라인 스타일 바인딩

- 개요 : 요소의 인라인 스타일을 동적으로 관리함
- 구현방법 : 객체 또는 배열 구문을 사용하여 스타일을 바인딩 함
